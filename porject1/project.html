무비 파인더

프로젝트 설명

영화를 찾고싶어 하는 사람들에게 빠르게 영화를 찾아주는 서비스

준비해야 할것

  • 프론트엔드에 사용될 HTML파일
  • 오픈소스로 제공되는 '오피스 박스API'
  • API의 정보를 담는 DB구축

개발 계획

  1. 메인 페이지 및 서브페이지 완성
    [서브 페이지]
    - 영화 상세 설명 페이지
    - 회원가입 페이지
  2. 오픈소스 API를 이용한 영화 데이터 이용
  3. 서버와 클라이언트간의 연동 확인

기능

  • 회원가입
  • 리뷰 및 별점 평가작성
  • 무비 파인더
